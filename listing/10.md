Что выведет данная программа и почему?

```
func update(p *int) {
	b := 2
	p = &b
}

func main() {
	var (
		a = 1
		p = &a
	)
	fmt.Println(*p)
	update(p)
	fmt.Println(*p)
}
```

Ответ:

```
    Вывод: 1 1
    Такой вывод потому что в функции update переменная p используется не по ссылке. Следовательно в основной функции значение не меняется.
    Для правильной работы программы необходимо использовать следующее выражение *p = b
```
