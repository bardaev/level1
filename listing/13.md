Что выведет данная программа и почему?

```
func someAction(v []int8, b int8) {
	v[0] = 100
	v = append(v, b)
}

func main() {
	var a = []int8{1, 2, 3, 4, 5}
	someAction(a, 6)
	fmt.Println(a)
}
```

Ответ:

```
    [100 2 3 4 5]
    
    Данная операция v[0] = 100 меняет значение по ссылке.
    v = append(v, b) возвращает новый слайс, который не используется
```
